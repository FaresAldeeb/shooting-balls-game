# 🎯 Shooting Balls Game

An interactive 2D shooting game developed using Python and Tkinter, demonstrating Human-Computer Interaction (HCI) principles and physics-based mechanics such as projectile motion and collision detection.

---

## 🧠 Overview
This project was developed as part of an HCI course to explore how users interact with graphical systems.  
The game combines interface design, real-time interaction, and basic physics to create an engaging user experience.

---

## 🚀 Features
- 🎮 Interactive 2D shooting gameplay  
- 🧠 Application of Human-Computer Interaction (HCI) principles  
- ⚙️ Physics-based projectile motion (angle, velocity, gravity)  
- 🎯 Collision detection between bullets and moving targets  
- 🎚️ Adjustable controls (angle, power, cannon position)  
- 📊 Real-time feedback (score, level, timer)  
- 🔄 Progressive difficulty with increasing levels  

---

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Math (trigonometry for motion calculations)

---
